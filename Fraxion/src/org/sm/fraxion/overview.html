Fraxion is a multi-threaded fractal exploration application, mainly providing Mandelbrot/Julia-type, Newton/Raphson-type, Magnet-type, and Markus/Lyapunov-type fractals.

This document contains the API documentation for all the Fraxion (version 1.3) classes.
The application was designed for the Java 2 Platform Standard Edition 1.8.0 or higher.
