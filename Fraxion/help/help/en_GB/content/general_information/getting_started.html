Getting started

Using Fraxion is fairly straightforward. The following examples show some of the possibilities that the application offers.

The application currently supports three languages: British English, American English, and Dutch. We can choose one of these languanges by specifying the -language=en_GB, -language=en_US, or - language=nl_BE parameter.

Select the Multibrot fractal from the Fractal menu, Specify fractal family and Powers submenus. Let us then pick an arbitrary power by selecting the Set power from the latter submenu. In the dialog box, specify 5.5 + 1.0 i.

Open the Color map menu, then enter the Specify exterior color map submenu and select the Hot color map. In the Color map menu, also enter the Setup interior color map submenu and select the Deep space color map from the Specify interior color map submenu. This leads to the following visualization:


Open the Color map menu, enter the Setup exterior color map submenu and select the Use striping option. From the Color map menu, enter the Setup interior color map submenu, and select the Use minimum distance to Gaussian integers. This modifies the visualization to the following:


Now let us look at the dual Julia set; make sure that the Show current location option is ticked in the Navigation menu. Position the mouse cursor at location 0.9215 + 0.7035 i in the complex plane. You can use the Magnifying glass from the Navigation menu to increase your pointing accuracy. Double- click the left mouse button. This gives you the following visualization:


Place the mouse cursor inside the top-right bulb, Click and hold the left mouse button. Then drag the mouse outward, until the selection rectangle is as large as shown in the following visualization:


We can see more details by selecting the Use rank order scaling option from the Color map menu, leading to the following visualization:

If you wish, you can enlarge the image using the Specify screen size option in the Navigation menu (see also Resizing for quality printing for more information). Finally, save your work with the Export to PNG option from the File menu.
