General commands

Shortcut Description
CTRL+AShow about box
CTRL+BUse binary decomposition
CTRL+CCopy coordinates at mouse cursor to clipboard
CTRL+DShow inset fractal
CTRL+ESet escape radius
CTRL+FUse a fixed exterior colour
CTRL+GShow magnifying glass
CTRL+HUse rank order colour map scaling
CTRL+ISet maximum number of iterations
CTRL+JReset to default colour map scheme
CTRL+KCycle colours
CTRL+LShow current location at mouse cursor
CTRL+MShow axes
CTRL+NShow orbit analyses
CTRL+OShow orbits
CTRL+PShow orbit paths
CTRL+QQuit application
CTRL+RReset zoom
CTRL+SSwitch main and dual fractal
CTRL+TScale orbits to window dimensions
CTRL+USet inset fractal size
CTRL+VZoom to level
CTRL+WUse contours
CTRL+XUse tiger stripes for exterior colouring
CTRL+YSet navigation panning size
CTRL+ZShow zoom information
F1Activate help

Mouse control

Shortcut Description
LEFT-CLICK + DRAGSpecify zoom rectangle/square
RIGHT-CLICKWhile selecting, disable current selection
DOUBLE-LEFT-CLICKSwitch fractal type / select orbit starting point
DOUBLE-RIGHT-CLICKZoom to previous level
SCROLLWHEELMove vertical scrollbar if present
